A mini API system to exemplify the basic structure of Java (Using Helidon)

</div>

## Run locally

### Prerequisites
- Java JDK
- Maven `mvn`

### Steps
1. `cd ./api`
2. `mvn package`
3. `java -jar target/api.jar`
